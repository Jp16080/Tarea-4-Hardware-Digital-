# Tarea-4-Hardware-Digital-
* **Módulo:**  ALU simple de 2 bits (AND, OR, XOR, suma)
* **Descripción:** Implementación de una Unidad Aritmética Lógica (ALU) de 2 bits simulada en CircuitVerse. El circuito realiza cuatro operaciones combinacionales (AND, OR, XOR y SUMA) controladas mediante un selector de 2 bits.
